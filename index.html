✈ FAST Update Generator — Help

🗂 How it works

  1. Admin setup: Click "Manage Team", set a Master PIN, and create profiles for all your engineers (Name + PIN personnel).
  2. Login: An engineer clicks his name and enters his PIN.
  3. Step 6: L'ingénieur connecté est automatiquement le "Lead Engineer". Il peut ajouter des collègues via le menu déroulant si besoin.
  4. Generate: The exact FAST table is created. SN ON and layouts are fully responsive.

💾 Data storage

All data is stored locally in your browser (localStorage). Nothing is sent to any server.
