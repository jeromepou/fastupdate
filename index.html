✈ FAST Update Generator — Help

🗂 How it works

  1. Admin setup: Click "Admin access" on the login screen. Sign in with your admin email/password (created in Firebase console). Then open "Manage Team" to create engineer profiles with PIN.
  2. Engineer login: Click your name → enter your 4-digit PIN.
  3. Cloud sync: All data is saved to Firebase in real time — accessible from any device anywhere.
  4. RTS projects: Mark a project RTS to move it to Archives — always accessible but separate from active projects.

🔒 Security

Data is stored in Firebase Realtime Database with authentication. Only verified users (admin + engineers via PIN) can access data. All traffic is encrypted (HTTPS).
